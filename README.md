# NODA — API

Бекенд для Q&A-платформи NODA — сервісу запитань і відповідей для студентів та розробників-початківців (аналог Stack Overflow / Quora).

---

## Можливості платформи

- Публікація запитань і відповідей
- Пошук за темами та тегами
- Лайки, репости, збереження постів
- Прийняття найкращої відповіді
- Профіль користувача з аватаркою та описом
- Підписка на теги
- Система ролей: студент → верифікований → модератор

---

## Технології

| Компонент | Технологія |
|---|---|
| Фреймворк | NestJS |
| Мова | TypeScript |
| База даних | PostgreSQL |
| ORM | Prisma |
| Пакетний менеджер | pnpm |
| Контейнеризація | Docker / docker-compose |
| Лінтер / форматер | ESLint + Prettier |

---

## Запуск

```bash
# Встановлення залежностей
pnpm install

# Запуск через Docker
docker-compose up --build

# Або локально
pnpm run start:dev
```

Необхідні змінні середовища (`.env`):
```env
DATABASE_URL=postgresql://user:password@localhost:5432/noda
JWT_SECRET=ваш_секрет
PORT=3000
```

API буде доступне за адресою `http://localhost:3000`
